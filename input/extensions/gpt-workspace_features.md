## Формулы GPT

```
=GPT("prompt", A1)           - результат в одну ячейку
=GPT_LIST("prompt", A1)      - результат списком по строкам
=GPT_TABLE("prompt", A1)     - результат таблицей
=GPT_FILE(file_url, "prompt") - извлечение из PDF/изображения
=GPT_TRANSLATE(A1, "ru")     - перевод текста
```
